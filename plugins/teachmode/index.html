Teachmode — Help & Info

Teachmode is a motion capture & playback tool for CNC machines and camera sliders. Record a sequence of positions, set timing, then play back for smooth, repeatable camera moves.

Jog Buttons

  • Tap an axis button (X+ / DRO) — moves 0.1 mm one step.
  • Hold — starts a continuous 500 mm/min jog; releases on pointer-up.
  • Feed Rate controls the jog speed.

Capturing Positions

  • Jog to the desired position, then press Capture.
  • The first capture is the Start Position — the machine moves here before playback begins.
  • Each subsequent capture records the position and automatically estimates travel time from the previous point using machine acceleration & max-rate data.
  • In Edit mode, pressing Capture updates the selected point's position (shown as Update).

Manual Time Editing

  • Each row shows a time field HH:MM:SS:ms — tap any segment to edit.
  • Changing the time automatically recalculates the feedrate (F) for that segment, so the machine will actually travel at that pace.
  • Use this to fine-tune shot timing independently of physical distance.

Edit Mode

  • Delete — removes the selected point.
  • Up / Down — reorder the selected point.
  • Done — exits edit mode.
  • Cancel — discards all edits since entering Edit mode.
  • Long-press any row (1 second) to instantly move the machine to that position.

Playback

  • Play — runs the sequence. Countdown starts once the machine reaches the first waypoint.
  • Pause — holds motion; press again to resume.
  • Abort — immediate stop and soft-reset.
  • Single — plays once. Repeat — loops continuously. Reverse — bounces forward then backward.

G-Code Field

Optional command sent alongside each captured move (e.g. M62 P0 to trigger a relay/camera shutter).

Time Prediction

Times are estimated using a trapezoidal motion profile — accounting for axis max-rate and acceleration from your FluidNC config file. Loaded automatically at startup. "No machine data" means defaults are in use; times will still show but may be less accurate.

Camera Slider Use

This plugin was designed for use with motorised camera sliders on FluidNC. Capture your start and end frames, set the total shot duration via the time fields, and let Teachmode drive the slider at a precisely timed, smooth speed — including ramp-up and ramp-down.
